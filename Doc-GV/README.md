# 🛡️ Doc-GV (Doc-Governance) v1.0.0

> **PII-Shield & End-to-End Governance Layer**

📌 데이터 리니지 추적과 **실시간 개인정보 보호(PII Protection)**를 통해  
안전하고 신뢰 가능한 **AX(AI Transformation) 환경**을 보장하는 거버넌스 모듈입니다.

Doc-GV는 문서 데이터가 **수집 → 가공 → 활용 → 반출**되는 전 과정에서  
보안, 감사, 책임성을 구조적으로 강제합니다.

---

## 🔒 개인정보 보호 및 보안 (PII Shield)

### 1. PII Detection
- 주민등록번호, 연락처, 주소, 계좌번호 등 **12종 이상의 개인정보 자동 탐지**
- 정규식 + 룰 기반 + 경량 NLP 조합 방식

### 2. Dynamic Masking
- 사용자 보안 등급(ACL)에 따라 **검색 결과 실시간 비식별화**
- 원본 데이터는 훼손하지 않고 뷰 레벨에서 마스킹 처리

### 3. Authoring Alert
- 문서 작성 및 수정 시 개인정보 포함 여부 실시간 감지
- 즉시 경고 및 **자동 마스킹 제안** 제공

---

## ✅ 주요 기능

### 데이터 리니지 관리
- 원본 → 정규화 → 색인 → 검색 → 내보내기
- 전 구간 데이터 흐름을 **Block 단위로 추적**

### 전방위 감사(Audit)
- 누가, 언제, 어떤 문서·개인정보에 접근했는지 상세 기록
- 검색·열람·내보내기 이벤트 전부 감사 로그화

### 내보내기 통제
- 대외 제출용 문서 생성 시
  - 자동 익명화
  - 정책 기반 필터링
- 원본 유출 없는 안전한 산출물 제공

## 🛠️ 환경 및 기술 기준

- **Baseline JDK:** Java 1.8  
- **Language:** Java (필수 아님, 정책/환경에 따라 대체 가능)
- **Encoding:** UTF-8
- **Storage:** RDB / Immutable Storage
- **Interface:** Internal REST API / Event Hook

> ⚠️ 본 프로젝트군은 **Java 1.8 기준 설계**를 원칙으로 하되,  
> 환경·정책·성능 요구에 따라 타 언어 또는 런타임으로 대체 가능합니다.
